WEEK 1 · DAYS 1–7
Python Core — "Make It Run"
Variables, loops, functions, conditionals, strings. Build CLI tools.
🎯 Number Guessing Game
Day 1–2 · Your first real Python program
BEGINNER ▼

What you build

  • Computer picks a random number 1–100
  • User guesses; game says "too high / too low"
  • Track number of tries, show score at end
  • Ask to play again

Concepts you'll learn

variables input() int() casting while loops if / elif / else random module print formatting
⚡ Bonus: Add a difficulty setting (easy = 10 tries, hard = 5 tries)
📝 Personal To-Do CLI App
Day 3–5 · A terminal app you'll actually use
BEGINNER ▼

What you build

  • Add, list, delete, and mark tasks as done
  • Save tasks to a .txt file so they persist
  • Menu-driven interface in terminal
  • Show tasks numbered with status (✓ / ✗)

Concepts you'll learn

lists dictionaries for loops functions file I/O open() f-strings enumerate()
⚡ Bonus: Add task priorities (high/medium/low) and sort by priority
🔐 Password Generator
Day 6–7 · Functions & string manipulation
BEGINNER ▼

What you build

  • User sets length and character types (symbols, numbers, uppercase)
  • Generate strong random passwords
  • Check password strength and give a score
  • Copy to clipboard (pyperclip library)

Concepts you'll learn

string module random.choices() function parameters return values pip install boolean flags
⚡ Bonus: Save generated passwords to an encrypted text file
WEEK 2 · DAYS 8–14
Data & Files — "Make It Useful"

Day 8–10 · CSV files & data analysis
INTERMEDIATE ▼

What you build

  • Log expenses with category, amount, date

csv module datetime list comprehensions dictionary grouping sorted() sum() / max() try / except

🌦️ Weather CLI Tool

INTERMEDIATE ▼

What you build

requests library JSON parsing API keys / .env HTTP status codes error handling python-dotenv

📰 News Headline Scraper

INTERMEDIATE ▼

What you build

  • Save them with timestamps to a JSON file

Concepts you'll learn

BeautifulSoup HTML parsing json module CSS selectors file writing string methods

WEEK 3 · DAYS 15–21
OOP & Real Apps — "Make It Scalable"
Classes, modules, databases. Build things with structure.
📚 Library Management System
Day 15–17 · Object-Oriented Programming deep dive
INTERMEDIATE ▼

What you build

  • Book class: title, author, ISBN, availability
  • Library class: add/remove/search/borrow books
  • Member class: track borrowed books & due dates
  • Save/load library state from JSON

Concepts you'll learn

classes & __init__ methods inheritance __str__ / __repr__ encapsulation class attributes multiple files
⚡ Bonus: Add overdue detection and calculate late fees
🗄️ Student Grade Tracker with SQLite
Day 18–19 · Your first real database
INTERMEDIATE ▼

What you build

  • Store students, subjects, and grades in SQLite
  • CRUD operations: add, view, update, delete
  • Calculate GPA per student and class average
  • Search and filter students by grade range

Concepts you'll learn

sqlite3 module SQL basics CREATE TABLE INSERT / SELECT WHERE / ORDER BY cursor & connection
⚡ Bonus: Export full report as a formatted PDF using reportlab
🤖 Telegram / Discord Bot
Day 20–21 · Automation that lives on the internet
ADVANCED ▼

What you build

  • Bot responds to commands like /weather, /joke, /todo
  • Connects your Week 2 weather tool to a real chat
  • Sends scheduled daily messages (news digest)
  • Runs 24/7 (deploy to free tier on Railway or Render)

Concepts you'll learn

python-telegram-bot async / await event handlers environment variables deployment basics requirements.txt
⚡ This is your first project running on the internet. Huge milestone.
WEEK 4 · DAYS 22–30
Capstone — "Make It Yours"
Pick one path. Build a portfolio-worthy project. Ship it.
🌐 Path A: Flask Web App
Day 22–30 · Build a URL shortener with a web UI
ADVANCED ▼

What you build

  • Paste a long URL → get a short code (like bit.ly)
  • Web interface with HTML forms, Flask routes
  • SQLite to store URLs + click count tracking
  • Dashboard showing your most-clicked links
  • Deploy live to Render (free)

Concepts you'll learn

Flask routes & views HTML templates (Jinja2) GET / POST SQLAlchemy basics deployment
📊 Path B: Data Analysis Project
Day 22–30 · Analyze a real-world dataset and tell a story
ADVANCED ▼

What you build

  • Pick a Kaggle dataset (Netflix, Spotify, COVID, etc.)
  • Clean and explore data with pandas
  • Create 5+ charts: trends, comparisons, distributions
  • Write a Jupyter Notebook report with insights
  • Publish on GitHub with a clear README

Concepts you'll learn

pandas matplotlib / seaborn DataFrame operations groupby() Jupyter Notebook data cleaning

Day 22–30 · Scripts that save you hours every week
ADVANCED ▼

What you build
